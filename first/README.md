## 4 注意
    4.1、由于要注册到网关，网关会根据url规则进行转发，因此本服务所有接口，需要统一使用一个与其他服务能区分开的标识
    4.2、由于注册到网关，网关会拦截权限，因此本服务的接口文档需要预留接收权限的参数，以便网关聚合文档时，可输入参数
